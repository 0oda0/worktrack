# Фирменный стиль МТУСИ → дизайн-система WorkTrack

> Источник: официальный брендбук — https://life.mtuci.ru/brandbook (изучен 2026-07-16,
> данные сняты со страницы: палитра, типографика, паттерны, иконки, кнопки).
> Это документ-анализ. Реализация — во фронтенд-спринтах, здесь только «как делать».

## Design Read

Внутреннее продуктовое приложение (учёт рабочего времени) для сотрудников университета.
Язык — доверительный, строгий, фирменный: белые поверхности, один акцент (фирменный
индиго МТУСИ), Montserrat, контурная графика. Не лендинг: никаких сценических
скролл-эффектов, декоративность — только в auth-экране и пустых состояниях.
Дайлы: VARIANCE 3–4, MOTION 2–3, DENSITY 4–5.

## 1. Что зафиксировано в брендбуке (факты)

### Палитра (официальная, 3 цвета)

| Роль | HEX | CMYK | Примечание |
|---|---|---|---|
| Основной цвет | `#372579` | 55/69/0/53 | глубокий индиго-фиолет, цвет логотипа и всей графики |
| Чёрный | `#151515` | 0/0/0/92 | не чистый #000 — фирменный почти-чёрный |
| Белый | `#FFFFFF` | 0/0/0/1 | доминирующий фон |

### Типографика

- **Единственный шрифт — Montserrat** (кириллица + латиница).
- Заголовки: **только Bold (700)**, кегль крупный («серьёзный, уверенный характер»).
- В логотипе шрифт искусственно растянут — в UI это не воспроизводим.

### Графический язык (что видно на странице брендбука)

- **Знак** — контурный семиугольник (heptagon) + словомарка «МТУСИ».
- **Паттерны** — тайлы из фрагментов букв/знака в трёх исполнениях: индиго-на-белом
  (почти незаметный, ~5% opacity), индиго-на-белом контрастный, белый-на-индиго.
  Правило построения: сетка 5×5, из которой можно вырезать 4×4, 3×3, 2×5 и т. д.
- **Кнопки на сайте брендбука** — pill-форма (полный радиус), белая заливка,
  тонкая обводка `#372579`, текст индиго, Bold.
- **Карточки** — белые, крупный радиус (~20–24px), тонкая индиго-обводка.
- **Иконки** — контурные (outline), единая толщина штриха ~2px, цвет индиго.
- **Иллюстрации** — line-art люди, индиго-контур + лавандовые заливки-акценты.
- Градиентов, теней и неона в стиле НЕТ — всё плоское, контрастное, контурное.

### Позиционирование

«МТУСИ — лидирующий университет в области информационных технологий,
телекоммуникаций, радиотехники и информационной безопасности».
Тон UI-текстов: уверенный, деловой, без канцелярита.

## 2. Токены WorkTrack (производные — помечено)

Официальных цветов три, поэтому промежуточные оттенки **выводим сами** от `#372579`
(это производные значения, не из брендбука):

```ts
// Mantine color scale 'mtuci' (0–9), primaryShade: 7 === #372579
const mtuci = [
  '#F2F0FA', // 0 — фон-подложка, hover строк таблиц
  '#E3DFF5', // 1 — выделение, badge-фоны
  '#C7BFE8', // 2
  '#A99CDA', // 3
  '#8C7ACC', // 4
  '#6F58BE', // 5
  '#55409E', // 6
  '#372579', // 7 — ФИРМЕННЫЙ (кнопки, ссылки, активные состояния)
  '#2B1D5E', // 8 — hover основного (тёмный тайл паттерна из брендбука)
  '#201545', // 9
]
```

Семантические статус-цвета (в брендбуке их нет — берём приглушённые, чтобы не спорили
с единственным акцентом): успех `green.7`, ошибка `red.7`, предупреждение `yellow.7`
из палитры Mantine; использовать только в бейджах/уведомлениях, не в декоре.

Текст: `#151515` основной, `#5C5872` (производный серо-фиолетовый) вторичный.
Фон приложения: белый; фон рабочей области за карточками: `#F8F7FC` (производный).

## 3. Тема Mantine (карта применения)

```ts
createTheme({
  primaryColor: 'mtuci',
  primaryShade: 7,
  colors: { mtuci },
  fontFamily: 'Montserrat, sans-serif',
  headings: { fontFamily: 'Montserrat, sans-serif', fontWeight: '700' },
  defaultRadius: 'lg',            // карточки ~16px
  components: {
    Button:      { defaultProps: { radius: 'xl' } },   // pill, как в брендбуке
    Badge:       { defaultProps: { radius: 'xl' } },
    Paper:       { defaultProps: { radius: 'lg', withBorder: true } },
    TextInput:   { defaultProps: { radius: 'md' } },
  },
})
```

- Шрифт Montserrat — self-host через `@fontsource/montserrat` (400/500/600/700),
  никаких `<link>` на Google Fonts.
- Иконки — `@tabler/icons-react`: контурные, stroke 2px — точное попадание в
  иконографику брендбука. Одна библиотека на проект, `stroke={1.75}` глобально.
- **Shape lock**: кнопки/бейджи — pill (xl), карточки/модалки — lg (~16px),
  инпуты — md (8px). Это правило, зафиксированное на весь проект.
- **Color lock**: один акцент `mtuci.7` на всё приложение. Никаких вторых акцентов,
  градиентов, свечений. Статус-цвета — только семантика.

## 4. Применение по экранам

| Экран | Приём |
|---|---|
| **Login / Register** | Белая pill-карточка по центру, фоном — фирменный паттерн из фрагментов знака (SVG с брендбук-страницы, opacity ~5%, как «светлый» тайл). Заголовок Montserrat Bold. Единственное место с декором. |
| **CheckIn (работник, мобайл)** | Одна крупная pill-кнопка «Пришёл»/«Ушёл» (заливка `mtuci.7` / обводка), статус геозоны бейджем, таймер смены крупным Bold с `font-variant-numeric: tabular-nums`. |
| **Timesheet** | Карточки-статистики: белые, индиго-обводка, число крупным Bold индиго, подпись `#5C5872`. Таблица дней: без вертикальных линий, hover `mtuci.0`, бейджи «выходной»/«вне зоны»/«ручная» — pill c фоном `mtuci.1`. |
| **Админка** | Сайдбар: белый, активный пункт — pill-заливка `mtuci.0` + текст `mtuci.7` (не инверсия). Заголовки разделов Bold. Таблицы плотные, числа tabular-nums. |
| **Графики (Recharts)** | Один цвет серии — `mtuci.7`, дополнительные серии — оттенки 4/2 той же шкалы. Сетка `#EEECF6`. Никакой радуги. |
| **Пустые состояния** | Контурная line-art пиктограмма в стиле иконок брендбука + короткий текст + одно действие. |

Логотип: использовать официальный SVG (знак-семиугольник + «МТУСИ») в шапке;
скачивается со страницы брендбука. Не перерисовывать, не менять пропорции —
брендбук прямо запрещает искажения.

## 5. Чего НЕ делать (анти-чеклист)

- Градиенты, glassmorphism, неоновые свечения — в стиле МТУСИ их нет.
- Второй акцентный цвет или «AI-фиолетовые» подмешанные оттенки вне шкалы `mtuci`.
- Чистый `#000000` — только фирменный `#151515`.
- Смешение форм: квадратные кнопки рядом с pill (см. shape lock).
- Тяжёлые анимации: максимум — 150–200ms переходы на hover/active и плавное
  появление контента; `prefers-reduced-motion` уважаем.
- Заголовки не-Bold или другим шрифтом.
- Паттерн — только на auth-экране и, дозированно, в пустых состояниях; не как фон
  рабочих таблиц.
